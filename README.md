# MBS Static Practice

Open `index.html` directly in a browser. No server, localhost, or LAN IP is required.

This version stores the practice match in the current browser with `localStorage`.

Use it for:

- one computer shared by four teammates;
- solo practice;
- independent practice on separate devices.

It does not synchronize decisions across different devices. For real-time multi-device collaboration, use the server version in `mbs-practice/`.
